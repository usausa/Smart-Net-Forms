# Smart.Forms .NET - MVVM helper library for Xamarin.Forms

## What does if contain ?

* Animation support.
* Platform abstraction.
* Basic converters.
* Observable commands. 
* Actions, Behaviors and Triggers.
* Markup extensions.
* Messenger.
* Resolver(DI Container) integration.
* Basic validation
* Base class for ViewModel.
